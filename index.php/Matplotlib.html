Importing for Use on Cluster Nodes[edit]

In order to run matplotlib inside a job running on a cluster node use the following code: 

import matplotlib
matplotlib.use('Agg')
import matplotlib.pyplot as plt

Example Script[edit]

This code generates the graph shown on the right. 

import numpy as np
import matplotlib.pyplot as plt

x = np.arange(0,5,0.1)
y1 = np.sin(x)
y2 = np.sin(3*x)
y3 = np.cos(3*x)

fig = plt.figure(figsize=(6,6))

sp = plt.subplot(3, 1, 1)
sp.plot(x, y1, "r-", label = "y1") 

sp.plot(x, y2, "bo", label = "y2")

sp.legend()

sp = plt.subplot(3, 1, 2)
sp.plot(x, y2, color = 'green', linestyle='solid', marker = '.', markerfacecolor = 'black', markersize = 12)

sp = plt.subplot(3, 1, 3)
sp.plot(x, y3, "k--")

sp.get_xlim()
sp.get_ylim()
sp.text(0.5, 0.5, "some text")

sp = plt.subplot(3, 1, 1)
plt.setp(sp.get_xticklabels(), visible=False)
sp = plt.subplot(3, 1, 2)
plt.setp(sp.get_xticklabels(), visible=False)
plt.setp(sp.get_yticklabels()[-1], visible=False)
sp = plt.subplot(3, 1, 3)
plt.setp(sp.get_yticklabels()[-1], visible=False)

fig.subplots_adjust(hspace=0)

sp=plt.subplot(3, 1, 2)
sp.set_ylabel("y1, y2, y3")

sp = plt.subplot(3, 1, 3)
sp.set_xlabel(r"$t / \tau$", fontsize=16) 

fig.savefig("matplotlibtest.png", dpi=300)

plt.show()

Sample matplotlib rc file[edit]

An easy way to make professional publication quality plots in matplotlib is to embed all necessary and repeatedly used settings (thicker lines, brighter markers, correct figure size, etc) in a single file called matplotlibrc and place it in either ~/config/matplotlib or ~/.matplotlib depending on your linux distro. Here's a sample: 

path.simplify: True
# Backends
backend: TkAgg # change to Agg if using on the cluster              		  
       
# Fonts 
font.family: serif
font.serif: ["Computer Modern"]
font.style: normal
font.variant: normal
font.weight: medium
font.stretch: normal
font.size: 10.0

# text
text.color: black
text.usetex: True
mathtext.default: regular
             
# Axes
axes.facecolor : white
axes.edgecolor: black
axes.formatter.use_mathtext : True
axes.titlesize: 10.0
axes.labelsize: 10.0
axes.xmargin: 0
axes.ymargin: 0
xtick.major.size : 4
ytick.major.size : 4
xtick.minor.size : 2
ytick.minor.size : 2
xtick.major.width : 1
ytick.major.width : 1
xtick.minor.width : 1
ytick.minor.width : 1
xtick.labelsize: 10
ytick.labelsize: 10

# Legends
legend.numpoints : 1
legend.frameon : False  
legend.handletextpad : 0.3
legend.fontsize : 10.0
legend.loc: best
legend.markerscale : 1
legend.labelspacing: 0.5

# Figures 
figure.autolayout: True
figure.facecolor : white
figure.edgecolor : white
figure.dpi: 150
savefig.dpi: 600
savefig.facecolor : white
savefig.edgecolor: white
savefig.format: png
savefig.bbox: tight
savefig.pad_inches: 0.04
savefig.directory: ./

# Lines
lines.linestyle: solid
lines.linewidth: 2.0
lines.markersize: 6
lines.color: black
lines.solid_capstyle: round
lines.solid_joinstyle: round
